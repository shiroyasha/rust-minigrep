# Minigrep

Minimal grep implementaion in Rust.

Follow this tutorial to get the same:
https://doc.rust-lang.org/book/second-edition/ch12-00-an-io-project.html.

## Build

```
cargo run
```

## Test

```
cargo test
```

## Search files

Usage:

```
cargo run <query> <file>
```

Example:

```
cargo run "Duck" poem.txt
```
